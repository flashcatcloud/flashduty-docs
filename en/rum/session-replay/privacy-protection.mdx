To meet privacy requirements for different scenarios, the session replay feature has built-in flexible privacy protection strategies. By configuring the `defaultPrivacyLevel` field, developers can control the sensitivity of data collection, supporting multiple modes from displaying all text (except passwords) to completely hiding page text, ensuring user data security and compliance.

<Warning>
Input fields with type password are sensitive information and **will not be collected in any scenario**.
</Warning>

## Privacy Strategies

<Tabs>
  <Tab title="Mask All Text">
    Configuring `defaultPrivacyLevel: "mask"` will completely hide all text content on the page, retaining only action behavior and page structure, suitable for scenarios with high data privacy requirements.

    ```javascript
    window.FC_RUM.init({
      applicationId: "YOUR_APPLICATION_ID",
      clientToken: "YOUR_CLIENT_TOKEN",
      // ...
      sessionReplaySampleRate: 10,
      defaultPrivacyLevel: "mask",
      // ...
    });
    ```

    <Frame>
      <img src="https://docs-cdn.flashcat.cloud/imges/png/3c123bbf8fd30482da766ab009c16b0e.png" alt="Mask All Page Text" />
    </Frame>
  </Tab>
  <Tab title="Mask User Input">
    Configuring `defaultPrivacyLevel: "mask-user-input"` will hide content in user input fields (such as text inputs, select boxes, etc.) but retain other page text, suitable for scenarios that need to protect user input privacy.

    ```javascript
    window.FC_RUM.init({
      applicationId: "YOUR_APPLICATION_ID",
      clientToken: "YOUR_CLIENT_TOKEN",
      // ...
      sessionReplaySampleRate: 10,
      defaultPrivacyLevel: "mask-user-input",
      // ...
    });
    ```

    <Frame>
      <img src="https://docs-cdn.flashcat.cloud/imges/png/5c43c6f6196a95334a8b253fa33360c9.png" alt="Mask Input Field Content" />
    </Frame>
  </Tab>
  <Tab title="Allow All Text">
    Configuring `defaultPrivacyLevel: "allow"` allows collection of all text content on the page except password fields, suitable for scenarios that need complete user interaction details.

    ```javascript
    window.FC_RUM.init({
      applicationId: "YOUR_APPLICATION_ID",
      clientToken: "YOUR_CLIENT_TOKEN",
      // ...
      sessionReplaySampleRate: 10,
      defaultPrivacyLevel: "allow",
      // ...
    });
    ```

    <Frame>
      <img src="https://docs-cdn.flashcat.cloud/imges/png/4cae182c91c7df152cbcf86c2978f443.png" alt="Show All Text" />
    </Frame>
  </Tab>
</Tabs>

## Configuration Comparison

| Configuration | Page Text | Input Content | Password Fields | Applicable Scenarios |
|---------------|-----------|---------------|-----------------|----------------------|
| `mask` | Hidden | Hidden | Hidden | High privacy requirement scenarios |
| `mask-user-input` | Shown | Hidden | Hidden | Protect user input |
| `allow` | Shown | Shown | Hidden | Complete interaction details |
