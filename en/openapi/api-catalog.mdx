Flashduty Open API provides **214** endpoints covering four major modules: On-call, Monitors, RUM, and Platform. All endpoints use unified authentication and request specifications. See [Quick Start](/en/openapi/introduction) for details.

<Note>
All endpoint URLs use `https://api.flashcat.cloud` as the base, authenticated via APP Key through query string.
</Note>
