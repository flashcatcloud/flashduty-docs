## On-Call Alert Management

A unified intelligent alert response platform: reduce noise, schedule, assign, escalate, and notify to help teams respond to and handle production incidents quickly.

<AccordionGroup>
  <Accordion title="Core Capabilities" icon="star">
    - **Intelligent noise reduction**: Alert grouping, inhibition, and deduplication to reduce 90% of alert noise
    - **Flexible assignment**: Multi-level escalation, dynamic routing, and rotation scheduling
    - **Multi-channel notifications**: Feishu/Lark, Dingtalk, WeCom, Slack, phone calls, SMS
    - **Rich integrations**: Native support for 100+ monitoring tools and cloud platforms
  </Accordion>
  
  <Accordion title="Use Cases" icon="bullseye">
    - Build 7x24 on-call systems to ensure service availability
    - Aggregate alerts from multiple sources into a unified incident management portal
    - Establish escalation mechanisms to ensure timely response
    - Analyze alert data to continuously improve monitoring quality
  </Accordion>
</AccordionGroup>

## RUM User Experience Monitoring

Real User Monitoring helps you understand how real users experience your application and quickly identify and resolve frontend issues.

<AccordionGroup>
  <Accordion title="Core Capabilities" icon="star">
    - **Performance monitoring**: Full-chain tracking of page loads, resource loads, and API calls
    - **Error tracking**: Automatic collection and grouping of JS errors and network errors
    - **Session replay**: Reproduce user operation paths to quickly replicate issues
    - **Custom metrics**: Report business-specific metrics to meet personalized needs
  </Accordion>
  
  <Accordion title="Use Cases" icon="bullseye">
    - Monitor core web vitals (LCP, FID, CLS) for web applications
    - Track and analyze frontend errors to improve application stability
    - Analyze user behavior paths to optimize product experience
    - Correlate with backend traces for full-stack observability
  </Accordion>
</AccordionGroup>

## Monitors Management

A unified monitoring management platform that aggregates data from multiple sources for full-stack observability.

<AccordionGroup>
  <Accordion title="Core Capabilities" icon="star">
    - **Multi-source aggregation**: Unified integration with Prometheus, Zabbix, cloud monitoring, and other data sources
    - **Unified view**: Cross-platform, cross-region unified monitoring dashboard
    - **Intelligent baselines**: Automatically calculate alert thresholds based on historical data
    - **Synthetic monitoring**: Global multi-node active probing of service availability
  </Accordion>
  
  <Accordion title="Use Cases" icon="bullseye">
    - Consolidate disparate monitoring systems into a unified observability portal
    - Build business-level monitoring dashboards to visualize system health
    - Implement proactive monitoring to detect potential issues early
    - Unified monitoring management across clouds and regions
  </Accordion>
</AccordionGroup>
