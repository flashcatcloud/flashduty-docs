## Overview

The rule template library provides a collection of predefined alert rule templates covering common monitoring scenarios (such as host monitoring, database monitoring, middleware monitoring, etc.). Templates allow you to quickly create alert rules, reducing the effort of configuring from scratch.

## Browse Templates

**Menu Entry**: Alert Rules → Template Library

Templates are organized by monitoring scenario category and support keyword search. Each template includes preconfigured query statements, threshold conditions, and recommended alert severity levels.

## Import a Template

<Steps>
  <Step title="Select a template">
    Browse the template library and find an alert rule template that fits your needs.
  </Step>
  <Step title="Configure parameters">
    Adjust parameters in the template according to your actual environment, such as data source, thresholds, notification methods, etc.
  </Step>
  <Step title="Create rule">
    After confirming the configuration, the system creates an alert rule based on the template and adds it to the specified rule group.
  </Step>
</Steps>

<Tip>
Importing a template creates an independent copy of the alert rule. You can freely modify it without affecting the original template.
</Tip>
