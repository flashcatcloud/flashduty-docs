## On-call 定价

---

On-call 采用 **License 订阅制**收费，按活跃用户数量计费。

### 收费模式

<Info>
每个 License 对应一个账户成员。只有持有 License 的成员才能使用 On-call 的全部功能。
</Info>

| 版本 | 说明 | 适用场景 |
| --- | --- | --- |
| **免费版** | 永久免费，功能受限 | 个人或小团队体验 |
| **标准版** | 基础功能完整 | 中小型团队日常使用 |
| **商业版** | 全部功能 + 高级特性 | 企业级生产环境 |

详细功能对比请访问：[价格页面](https://flashcat.cloud/flashduty/price/)

### License 类型

<Tabs>
  <Tab title="固定 License">
    - 在购买有效期内**长期有效**
    - **不会被抢占**
    - **管理者** 可以授予成员固定 License
    - 适用于需要参与处理故障、配置业务的成员
  </Tab>
  <Tab title="临时 License">
    - 在购买有效期内**长期有效**
    - 每个周期结束时**自动释放**
    - **管理者** 可以授予成员临时 License
    - 在有足够 License 时，可通过分配或抢占方式占用
    - 适用于临时参与或偶尔使用的成员
  </Tab>
</Tabs>

### 无 License 成员权限

<Warning>
没有 License 的成员功能受限，仅能被动接收告警消息。
</Warning>

| 能力 | 说明 |
| --- | --- |
| 查看故障 | ❌ 不可查看故障列表/详情 |
| 处理故障 | ❌ 不可认领、关闭等操作 |
| 接收通知 | ✅ 可被动接收告警消息 |
| 通知额度 | ✅ 共享租户的邮件、短信、电话套餐额度 |

## RUM 定价

---

RUM 采用 **按量付费**模式，根据实际使用的会话数量计费。

### 版本对比

  <Card title="专业版" icon="rocket">

  </Card>

### 功能与定价明细

| 功能 | 免费版 | 专业版 |
| --- | --- | --- |
| **会话分析**（视图、资源、异常等） | 每应用 1,000 次/月 | 无限制，¥9.0 / 千次会话 |
| **会话重放**（用户行为记录） | 每应用 500 次/月 | 无限制，¥12.0 / 千次会话 |
| **Web 性能监控** | ❌ | ✅ |
| **AI 错误追踪** | ❌ | ✅ |
| **高级预置看板** | ❌ | ✅ |
| **前后端关联**（Tracing） | ❌ | ✅ |

<Tip>
免费版适合评估产品功能，专业版按实际用量计费，无需预付大额费用。
</Tip>

## 常见问题

---

<AccordionGroup>
  <Accordion title="On-call 如何计算活跃用户？">
    当月使用商业化功能的用户即为活跃用户。每个月度周期结束后：
    - 固定 License 保持有效
    - 临时 License 自动释放
    - 成员被删除时，其 License 自动释放
  </Accordion>

  <Accordion title="没有 License 的成员能做什么？">
    - **可以**：被动接收告警消息，共享租户通知额度
    - **不可以**：查看/处理故障，进行任何平台配置操作
    
    在分派策略中可以选择将故障通知给没有 License 的成员，但该成员无法对故障进行操作。
  </Accordion>

  <Accordion title="RUM 会话如何计算？">
    - **会话分析**：每次用户访问应用的完整会话，包含页面浏览、资源加载、异常等数据
    - **会话重放**：记录用户操作行为的会话，用于回放用户交互过程
    
    两者独立计费，按实际产生的会话数量统计。
  </Accordion>

  <Accordion title="是否支持私有化部署？">
    支持。Flashduty 提供与 SaaS 服务几乎一致的私有化版本。
    
    <Warning>
    私有化部署有较高的维护成本，收费模式与 SaaS 不同。如无特殊需求，推荐使用云服务。
    </Warning>
    
    如需私有化版本，请联系我们。
  </Accordion>
</AccordionGroup>
