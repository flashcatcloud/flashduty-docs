## 什么是 MCP

[Model Context Protocol (MCP)](https://modelcontextprotocol.io/introduction) 是一种开放协议，用于在 AI 应用与外部工具/数据源之间建立标准化的通信接口。通过 MCP，AI 助手可以安全地调用外部系统的 API，获取实时数据并执行操作，而无需针对每个系统编写定制化的集成代码。

## 什么是 Flashduty MCP Server

Flashduty MCP Server 是基于 MCP 协议构建的服务，提供与 Flashduty API 的无缝对接。它将 Flashduty 的核心能力封装为一组标准化工具，使 AI 助手能够直接管理故障、查询变更、操作状态页等。

### 核心能力

Flashduty MCP Server 提供 **6 大工具集、18 个工具**，覆盖故障管理的完整生命周期：

| 工具集 | 说明 | 工具数 |
| --- | --- | --- |
| `incidents` | 故障生命周期管理（查询、创建、更新、认领、关闭、相似故障检索） | 8 |
| `changes` | 变更记录查询（部署、配置变更等） | 1 |
| `status_page` | 状态页管理（查询、变更事件、创建故障、更新时间线） | 4 |
| `users` | 成员和团队查询 | 2 |
| `channels` | 协作空间和分派策略查询 | 2 |
| `fields` | 自定义字段定义查询 | 1 |

### 应用场景

- **自动化故障响应**：AI 助手自动识别并认领故障，查找相似历史故障辅助排查
- **变更关联分析**：将故障与近期变更记录关联，快速定位根因
- **状态页维护**：通过自然语言指令创建状态页故障并更新进展
- **团队协作查询**：快速查找成员、团队和协作空间信息
- **数据提取与报告**：从 Flashduty 抽取故障数据，生成分析报告

## 支持的 AI 工具

Flashduty MCP Server 兼容所有支持 MCP 协议的 AI 工具和客户端：

| AI 工具 | 部署方式 | 说明 |
| --- | --- | --- |
| [Claude Desktop](https://claude.ai) | 本地 (stdio) | Anthropic 官方桌面客户端 |
| [Cursor](https://www.cursor.com/) | 远程 / 本地 | AI 驱动的代码编辑器 |
| [Windsurf](https://www.codeium.com/windsurf) | 本地 (stdio) | Codeium 推出的 AI IDE |
| [Claude Code](https://docs.anthropic.com/en/docs/claude-code) | 本地 (stdio) | Anthropic 命令行 AI 工具 |
| 其他 MCP 客户端 | 远程 / 本地 | 任何遵循 MCP 协议的客户端均可接入 |

<Note>
远程部署方式要求 MCP 客户端支持远程 MCP 服务器连接（Streamable HTTP 传输）。如果您的客户端不支持远程连接，请使用本地部署方式（stdio 传输）。
</Note>

## 架构概览

Flashduty MCP Server 支持两种运行模式：

```
┌─────────────────────────────────────────────────────┐
│                    AI 工具 / 客户端                    │
│         (Claude Desktop, Cursor, Windsurf 等)         │
└──────────────┬──────────────────────┬────────────────┘
               │                      │
         远程 (HTTP)             本地 (stdio)
               │                      │
               ▼                      ▼
┌──────────────────────┐  ┌──────────────────────────┐
│   远程 MCP Server     │  │    本地 MCP Server        │
│ mcp.flashcat.cloud   │  │  Docker / 二进制文件        │
└──────────┬───────────┘  └────────────┬─────────────┘
           │                           │
           └───────────┬───────────────┘
                       │
                       ▼
            ┌─────────────────────┐
            │   Flashduty API     │
            │ api.flashcat.cloud  │
            └─────────────────────┘
```

- **远程模式**：直接连接 Flashduty 托管的 MCP 服务 `mcp.flashcat.cloud/mcp`，无需安装任何组件
- **本地模式**：通过 Docker 或二进制文件在本地运行 MCP Server，通过 stdio 与 AI 工具通信
