状态页是你向用户展示服务状态的统一窗口。本文介绍如何创建状态页，以及如何配置状态页的基础属性和显示设置。

## 创建状态页

<Steps>
  <Step title="进入状态页管理">
    在 Flashduty 控制台左侧导航中，进入 **状态页** 管理页面，点击 **创建状态页**。
  </Step>
  <Step title="填写基本信息">
    配置以下必填字段：

    | 字段 | 说明 |
    | --- | --- |
    | **名称** | 状态页的显示名称，通常为公司或产品名称 |
    | **URL 标识** | 用于构建状态页访问地址的唯一标识，仅允许小写字母、数字和连字符 |
    | **类型** | 选择 **公开**（Public）或 **内部**（Internal） |
  </Step>
  <Step title="完成创建">
    点击 **确认** 完成创建。创建后，你可以继续配置组件、事件模板和订阅等高级设置。
  </Step>
</Steps>

### 状态页类型

| 类型 | 访问方式 | 订阅方式 | 适用场景 |
| --- | --- | --- | --- |
| **公开** | 公网可访问，无需登录 | 邮件订阅 | 面向客户和合作伙伴 |
| **内部** | 需 Flashduty 账号登录 | IM 集成推送 | 面向组织内部员工 |

<Note>
公开状态页不支持 IM 订阅，内部状态页不支持邮件订阅。
</Note>

### URL 标识

URL 标识是状态页访问地址中的唯一路径段。创建后可以修改，但同一账户内同类型状态页的 URL 标识不能重复。

系统会将名称自动转换为 URL 安全格式（小写化、非字母数字字符替换为连字符）。你也可以手动指定。

## 外观设置

你可以自定义状态页的品牌外观，使其与你的企业形象保持一致。

### Logo 与图标

| 设置项 | 说明 |
| --- | --- |
| **Logo** | 状态页顶部展示的品牌标识 |
| **暗色 Logo** | 在暗色模式下展示的品牌标识。未设置时默认使用主 Logo |
| **Logo 链接** | 用户点击 Logo 后跳转的目标地址 |
| **Favicon** | 浏览器标签页中显示的小图标 |

### 页面内容

| 设置项 | 说明 |
| --- | --- |
| **页头信息** | 状态页顶部展示的自定义文本 |
| **页脚信息** | 状态页底部展示的自定义文本 |
| **自定义链接** | 在状态页中展示额外的导航链接，方便用户快速跳转到相关资源 |
| **联系信息** | 获取帮助的联系方式，支持邮件地址或网站链接 |

## 自定义域名

默认情况下，状态页通过 Flashduty 提供的子域名访问。如果你希望使用自有域名（如 `status.yourcompany.com`），可以配置自定义域名。

<Steps>
  <Step title="添加 CNAME 记录">
    在你的 DNS 服务商中，为目标域名添加一条 CNAME 记录，指向 Flashduty 提供的状态页地址。
  </Step>
  <Step title="配置自定义域名">
    在状态页设置中填入自定义域名。系统会验证域名唯一性——同一域名不能被多个状态页使用。
  </Step>
</Steps>

<Warning>
自定义域名在全平台范围内必须唯一。如果该域名已被其他状态页使用，系统会拒绝设置。
</Warning>

## 显示设置

### 事件历史视图

状态页支持两种事件历史的展示方式：

| 视图模式 | 说明 |
| --- | --- |
| **日历视图** | 按日历形式展示历史事件，便于查看某一天的服务状态 |
| **列表视图** | 按时间线列出历史事件，适合快速浏览近期事件 |

### 可用性统计展示

你可以控制组件可用性统计信息的展示方式：

| 展示模式 | 说明 |
| --- | --- |
| **图表与百分比** | 同时展示可用性趋势图和百分比数值 |
| **仅图表** | 仅展示可用性趋势图 |
| **不展示** | 隐藏可用性统计信息 |

## 订阅设置

在状态页设置中，你可以启用或关闭订阅功能：

- **公开状态页**：支持启用**邮件**订阅。启用后，访客可以在状态页上通过邮件订阅事件更新。
- **内部状态页**：支持启用 **IM** 订阅。启用后，组织内的用户可以通过已绑定的 IM 集成接收推送通知。

### 事件模板偏好

你可以为状态页选择默认的事件模板类型，在发布事件时自动使用对应类型的模板。支持的模板类型包括：

- **预定义模板**：包含标题、描述和状态的完整事件模板
- **消息模板**：针对每个事件状态预设更新消息的模板

详见[事件模板](/zh/on-call/statuspage/templates)。

---

## 编辑与删除

你可以随时编辑状态页的名称、URL 标识、外观设置和显示配置。修改 URL 标识后，原访问地址将失效。

删除状态页将同时清除其下的所有组件、事件和订阅数据，此操作不可恢复。
