## 告警列表

告警管理页面提供了独立于故障的告警视角，你可以在此查看所有告警的详细信息，无论这些告警是否已被聚合到故障中。

进入 **故障列表** → **告警列表**，即可访问告警管理页面。

告警列表展示以下信息：

| 列 | 说明 |
| :--- | :--- |
| **严重程度** | 以色条形式展示告警的严重程度（Critical、Warning、Info） |
| **标题** | 告警的标题描述 |
| **处理状态** | 告警当前是否已关闭（已关闭 / 未关闭） |
| **恢复状态** | 已关闭的告警是否自动恢复（已恢复 / 未恢复） |
| **持续时间** | 告警从触发到当前或关闭的持续时间 |
| **关联事件数** | 告警关联的原始事件数量 |
| **关联故障** | 告警所属的故障，点击可跳转到故障详情 |
| **触发时间** | 告警首次触发的时间 |
| **集成来源** | 告警来源的集成名称和类型（共享或专属） |
| **协作空间** | 告警所属的协作空间 |

<Note>
被静默或抑制的告警会在标题前显示特殊标记，帮助你快速识别。
</Note>

### 筛选与搜索

告警列表顶部提供丰富的筛选能力，与故障列表类似，支持按时间范围、严重程度、标签等条件进行筛选。筛选条件保存在本地存储中，刷新页面后依然保留。

### 批量操作

你可以在列表中勾选多条告警进行批量操作，如批量关闭等。

### 使用聚合视图

除了标准的列表视图，告警列表还支持 **聚合视图**。聚合视图将告警按指定维度分组显示，左侧展示分组卡片列表，右侧展示选中分组下的告警详情。

<Steps>
<Step title="创建聚合视图">
切换到聚合视图后，如果尚未创建聚合规则，系统将引导你创建第一个聚合视图。
</Step>
<Step title="选择分组维度">
选择用于分组的聚合规则，系统将按该维度对告警进行分组。
</Step>
<Step title="浏览分组">
左侧面板展示各分组卡片，包含分组标题、严重程度和告警数量。点击分组卡片，右侧列表展示该分组下的所有告警。
</Step>
</Steps>

<Note>
在聚合视图下，系统最多匹配 **100 条** 数据进行分组展示。如果需要查看全量数据，请切换为列表视图。
</Note>

## 告警详情

点击告警标题进入告警详情页。告警详情页提供三个 Tab：

| Tab | 说明 |
| :--- | :--- |
| **告警概览** | 展示告警描述、标签、属性信息（协作空间、持续时间）、图片、告警状态时间线（触发和关闭时间）、告警来源、所属故障以及 Links |
| **时间线** | 展示告警生命周期中的所有事件记录 |
| **关联事件** | 展示该告警关联的所有原始事件列表 |

告警详情页顶部展示告警的关键信息，包括标题、严重程度、处理状态、短 ID 等，并提供关闭、并入等操作按钮。

<Tip>
你也可以在告警列表的视图设置中，选择以 **右侧面板** 模式查看详情，无需跳转新页面即可快速浏览告警信息。
</Tip>
