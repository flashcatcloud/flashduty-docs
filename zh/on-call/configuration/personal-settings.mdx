## 消息通知

---

按告警级别配置不同的通知方式。系统支持为 **Critical（紧急）**、**Warning（警告）** 和 **Info（提醒）** 三个级别分别设置通知渠道。

你可以为每个级别选择以下通知渠道：

| 渠道类型 | 说明 |
| --- | --- |
| **语音** | 通过电话语音通知（私有化部署可能不支持） |
| **短信** | 通过短信通知 |
| **邮件** | 通过邮箱发送通知 |
| **Flashduty APP** | 通过 Flashduty 移动端应用推送通知 |
| **IM 应用** | 通过已集成的钉钉、飞书、企业微信、Slack 或 Microsoft Teams 应用发送通知。你可以选择具体的 IM 集成实例 |

<Tip>
仅当满足以下条件时，才会按个人偏好通知：
- 故障直接分派到个人，或
- 分派策略设置为"单聊 + 遵循个人偏好"
</Tip>

如需使用钉钉、企业微信、飞书等 IM 通知，请先在对应应用中完成账号关联。每个级别可以同时选择多个通知渠道，确保重要告警不被遗漏。

## APP Key

---

APP Key 用于 API 请求认证。

| 限制 | 说明 |
| --- | --- |
| **数量上限** | 每个账号最多 5 个 |
| **权限范围** | 拥有全部 API 操作权限 |
| **安全提示** | 仅创建时显示，请妥善保存 |

<Warning>
- APP Key 泄露可能导致数据安全风险，请务必保密
- 删除前确认无业务依赖，删除后引用该 Key 的业务将立即失效
</Warning>

## Flashduty APP

---

### 下载应用

| 平台 | 下载方式 |
| --- | --- |
| **iOS** | App Store 搜索"Flashduty" |
| **Android** | 已上架小米、华为、荣耀、OPPO、vivo 等主流应用市场，搜索"Flashduty"即可下载（暂不支持鸿蒙） |

如果您使用的手机品牌不在以上列表中，可以在 APP 管理页面点击**点此下载**获取安装包二维码，使用手机扫描即可下载。

### 设备关联

下载安装后，您需要将移动设备与 Flashduty 账号关联，以接收推送通知。

<Steps>
<Step title="进入 APP 管理页面">
前往 **个人中心 → Flashduty APP** 页面。
</Step>
<Step title="扫码关联设备">
页面会显示一个二维码。打开 Flashduty APP，点击「扫码登录」按钮，扫描页面上的二维码即可完成设备关联。
</Step>
</Steps>

关联成功后，页面会展示已关联的设备列表，包括设备名称、操作系统版本和设备标识。如需解除关联，将鼠标悬停在设备卡片上，点击**解除关联**按钮即可。

<Note>
- 关联设备不等同于开启通知，故障的具体通知方式取决于您的分派策略设置
- 解除设备关联后，该设备将无法接收推送消息
- 二维码具有时效性，过期后页面会自动刷新生成新的二维码
</Note>

### 推送通知

Flashduty APP 支持免费推送告警通知，确保关键告警能即时触达。您可以在移动端随时随地认领、解决和升级告警。建议开启系统通知权限，并在勿扰模式中放行紧急故障通知，以免错过重要告警。

## 常见问题

---

<AccordionGroup>
  <Accordion title="设置了个人偏好通知方式，为什么没有触发通知？">
    1. 检查协作空间是否配置了分派策略。未配置分派策略时，故障不会触发通知
    2. 检查分派策略的单聊渠道设置。仅"遵循个人偏好"模式下才会使用您的个性化配置；"遵循统一设定"模式下将忽略个人偏好
    
    进入 协作空间详情 → **分派策略** 查看具体设定。
  </Accordion>
</AccordionGroup>
