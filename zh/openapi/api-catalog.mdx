Flashduty Open API 共提供 **214** 个接口，覆盖 On-call、Monitors、RUM 和平台四大模块。所有接口使用统一的认证方式和请求规范，详见[快速入门](/zh/openapi/introduction)。

<Note>
所有接口 Endpoint 均为 `https://api.flashcat.cloud`，使用 APP Key 通过 query string 认证。
</Note>
