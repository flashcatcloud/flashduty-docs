## 概述

API Key 用于 `monitedge` 告警引擎与 Flashduty SaaS 端之间的身份认证和安全通信。每个引擎在注册时需要提供有效的 API Key，SaaS 端据此验证引擎身份并建立通信通道。

**菜单入口**：告警引擎 → API Key 管理

## 创建 API Key

<Steps>
  <Step title="进入 API Key 管理页面">
    在告警引擎菜单下，点击 API Key 管理。
  </Step>
  <Step title="创建新 Key">
    点击创建按钮，填写 Key 名称（用于标识用途），系统自动生成唯一的 API Key。
  </Step>
  <Step title="保存 Key">
    API Key 仅在创建时完整展示一次，请妥善保存。
  </Step>
</Steps>

## 管理 API Key

| 操作 | 说明 |
|------|------|
| 重命名 | 修改 API Key 的显示名称，不影响 Key 值本身 |
| 启用/禁用 | 禁用后，使用该 Key 的引擎将无法与 SaaS 端通信 |
| 删除 | 永久删除 API Key，关联的引擎需更换为其他有效 Key |

<Warning>
禁用或删除正在使用中的 API Key 会导致对应的告警引擎失去连接，请确认无引擎在使用该 Key 后再操作。
</Warning>
